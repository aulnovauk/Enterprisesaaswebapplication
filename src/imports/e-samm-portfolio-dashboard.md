Design a full enterprise SaaS Portfolio Dashboard 
for a Solar Asset Management & Monitoring Platform (E-SAMMP).

This is a PSU-grade governance dashboard.
Not a marketing page.

Theme:
- Navy left sidebar
- White workspace
- Green = compliant
- Red = non-compliant
- Yellow = warning
- Compact enterprise layout
- Data-dense design

Portfolio Size:
220 MW
45 Plants
3 States (Maharashtra, Tamil Nadu, Goa)

==================================================
GLOBAL TOP FILTER BAR
==================================================

Include:
- Financial Year dropdown
- Month selector
- State filter
- Vendor filter
- PPA Type filter
- Duration toggle (Monthly / MTD / YTD / Annual)
- Export button
- Last updated timestamp

==================================================
ROW 1 – STRATEGIC KPI SUMMARY CARDS
==================================================

Create 8–10 horizontal KPI cards:

1. Total Installed Capacity (MW)
2. MTD Generation (MWh)
3. YTD Generation
4. Portfolio CUF (%)
5. Grid Availability (%)
6. Revenue Realized (₹)
7. Revenue Shortfall (₹)
8. Total LD Exposure (₹)
9. Asset Health Index (Score 0–100)
10. CO₂ Emission Reduction (Tonnes)

Each card must show:
- Current value
- Target value
- MoM change %
- Trend sparkline
- Compliance color indicator

==================================================
ROW 2 – GEOGRAPHICAL & RISK VIEW
==================================================

Left Side:
Interactive India map with plant markers.

Marker colors:
- Green = compliant
- Yellow = warning
- Red = non-compliant
- Grey = shutdown
- Orange = curtailment affected

Right Side:
Risk & Alert Panel:

- Non-Compliant Plants count
- High LD Risk Sites
- Escalation triggered cases
- Pending JMR submissions
- Top underperforming plants

==================================================
ROW 3 – GENERATION & PERFORMANCE ANALYTICS
==================================================

Left:
MTD vs Target Generation (Bar chart)

Middle:
CUF Trend (12 Month Line Chart)

Right:
Downtime Classification (Donut Chart)
- Grid outage
- Equipment failure
- Planned shutdown
- Curtailment
- Force majeure

==================================================
ROW 4 – COMMERCIAL & FINANCIAL INSIGHT
==================================================

Left:
Revenue Waterfall Chart
Budgeted → Expected → Actual → Realized

Middle:
LD Exposure (Vendor-wise Table)

Right:
O&M Deviation Settlement Panel
- PR Benchmark
- Actual PR
- Settlement Amount (₹)

==================================================

Left:
Vendor Performance Ranking Table
Columns:
- Vendor Name

- Avg Availability
- LD Exposure
- Compliance %

Right:
Cluster Comparison Chart
- Maharashtra
- Tamil Nadu
- Goa

==================================================
ROW 6 – ADVANCED ANALYTICS (OPTIONAL)
==================================================

Left:
Lost Production Index (LPI) chart

Right:
Asset Health Index breakdown
- PR Score
- Availability Score
- Downtime Score
- Compliance Score

==================================================
DRILL-DOWN FUNCTIONALITY
==================================================

Clicking:
- Any KPI card → Open KPI detail
- Any plant on map → Open Site Dashboard
- Any vendor → Open Vendor performance page

==================================================
DESIGN RULES
==================================================

- 12-column grid layout
- No excessive white space
- Compact tables
- Professional typography
- Rupee symbol ₹ for all financial values
- Realistic dummy data
- PSU-grade seriousness

The final dashboard should look like 
a Centralized Solar Asset Governance Command Center,
not a startup analytics page.
